## **Desafío 1 - Alura Store**

En este desafío tuve un primer acercamiento al análisis de datos. Se me proporcionaron cuatro datasets con información referente a las ventas de cuatro tiendas.

El objetivo era identificar qué tienda tenía el menor desempeño en ventas para poder recomendar la venta de la misma ya que el dueño deseaba invertir ese dinero en un nuevo emprendimiento.

Para realizar el desafío **simulando un entorno de trabajo profesional** se nos prporcionó además un [tablero de control](https://trello.com/b/38g2ykhd/alura-store-latam) en la plataforma **Trello**. 

## **Tecnologías utilizadas**

Los archivos fueron proporcionados en formato `.csv`

Se utilizó el lenguaje de programación **Python** usando la plataborma [**Colab**](https://colab.research.google.com/) de **Google** por lo que todo el trabajo se realizó en un notebook interactivo único.

El análisis se realizó con la biblioteca [**Pandas**](https://pandas.pydata.org/).

## Dependencias externas
Solo se realizó la importación de la biblioteca **Pandas** utilizando el siguiente comando:
